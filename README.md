# My fork of dmenu

Forked from [https://git.suckless.org/dmenu](https://git.suckless.org/dmenu) with the following patches:

* [line height](https://tools.suckless.org/dmenu/patches/line-height)
* [xyw](https://tools.suckless.org/dmenu/patches/xyw)

## Requirements

To build dmenu, the libX11 header files are required.

## Installation

```
git clone https://github.com/icanalesm/dmenu.git
cd dmenu
make
sudo make install
```
